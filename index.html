League Stats Bot

A Discord bot developed in JavaScript using the discord.js library. A user can execute commands and the bot replies with either a user's League of Legends account information such as their rank, champion statistics or their most recent match. It could also be used to look up other players and professional players and their stats.

The bot also has some basic utility commands which the user can user to learn more about the server which the bot is in or information about any other member in the server.

We Split

WeSplit is a web app which allows users to organize their bills easily. WeSplit allows users to create their own groups and then add different bills to each group which the group members can access.

Users can then settle their debt on the website by entering how much they paid each time. Each user also has an account, allowing a user's actitivity to be easily stored each time.

Aim Haskell

Aim Haskell is an aim trainer made in Haskell, primarily using the gloss library. The game emulates an aim trainer's core feature, allowing the user to train different aspects of their aim from precision to flicking.

The game also includes a scroing system where if a user manages to achieve a high score then they can save their score along with their name.

The purpose of this project was to further understand and learn the functional paradigm, with the main aim being exploring what monads are and the various ways to make use of them.

CL Pad

CL Pad is a command line text editor created using C. The program allows the user to perfrom basic line operations in a file and also perform operations related to files such as creating, appending and deleting files.
